# CPO (Chief Procurement Officer) Journey Matrix

**Persona:** Chief Procurement Officer  
**Primary Goal:** Validate $1.3M-$1.4M annual ROI, strategic oversight  
**Demo Priority:** CRITICAL (Segment 1 focus)

---

## Journey Timeline

### Phase 1: Arrival (0-30 seconds)
**User Action:** Opens Velocity URL in browser

| Step | Expected Outcome | Validation |
|------|------------------|------------|
| Browser loads | Login page appears | ☐ No errors |
| See branding | Velocity logo, professional styling | ☐ Premium feel |
| Find login | Email/password fields visible | ☐ Immediate access |

**CPO Internal Thought:** "This looks professional. Where do I sign in?"

---

### Phase 2: Authentication (30-60 seconds)
**User Action:** Enters credentials, clicks Sign In

| Step | Expected Outcome | Validation |
|------|------------------|------------|
| Enter email | Field accepts input | ☐ No issues |
| Enter password | Characters masked | ☐ Security visible |
| Click Sign In | Loading indicator briefly | ☐ < 2 seconds |
| Redirect | Executive Dashboard appears | ☐ Correct landing |

**CPO Internal Thought:** "Good, I'm in. What am I looking at?"

---

### Phase 3: First Impression (1-2 minutes)
**User Action:** Scans dashboard for value

| Step | Expected Outcome | Validation |
|------|------------------|------------|
| See metrics | ROI cards visible ($1.3M-$1.4M) | ☐ Numbers prominent |
| Find alerts | AI Insights section populated | ☐ Actionable items |
| Check budget | Budget health indicators | ☐ Visual clarity |
| View trends | Charts show real data | ☐ Not mock |

**CPO Internal Thought:** "I can see the value immediately. What's at risk?"

---

### Phase 4: Deep Dive (2-5 minutes)
**User Action:** Explores specific areas of concern

| Navigation | Purpose | Expected |

**CPO Internal Thought:** "This tells me what I need to know before meetings."

### Phase 6: Exit (10+ minutes)

## Pain Points Solved (Must Demonstrate)

1. **"Where's my money going?"** → Budget health dashboard, variance alerts
2. **"What should I worry about?"** → AI Insights with prioritized risks
3. **"How do I know if we're compliant?"** → Compliance dashboard, status badges
4. **"Can I trust this data?"** → Calculation formulas visible, audit trail

---

## Red Flags to Avoid

- ❌ Login doesn't work
- ❌ Dashboard shows "0" or "N/A" values
- ❌ "Demo", "Mock", "Sample" text visible
- ❌ Console errors in browser
- ❌ Slow load times (> 3 seconds)
- ❌ Broken navigation links
- ❌ Charts with no data
