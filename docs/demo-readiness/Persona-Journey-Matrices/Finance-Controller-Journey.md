# Finance Controller Journey Matrix

**Persona:** Finance Controller  
**Primary Goal:** Invoice accuracy, variance detection, budget compliance  
**Demo Priority:** CRITICAL (Invoice Variance is key differentiator)

---

## Journey Timeline

### Phase 1: Arrival (0-30 seconds)
**User Action:** Opens Velocity URL in browser

| Step | Expected Outcome | Validation |
|------|------------------|------------|
| Browser loads | Login page appears | ☐ Premium design |
| Enter credentials | Email/password accepted | ☐ Quick access buttons |
| Sign in | Dashboard redirect | ☐ < 2 seconds |

**Finance Controller Internal Thought:** "Professional login. Let me see the numbers."

---

| Step | Expected Outcome | Validation |

## Pain Points Solved (Must Demonstrate)

1. **"I spend hours hunting for billing errors"** → Variance detection highlights them automatically
2. **"I don't know we're over budget until it's too late"** → Proactive threshold alerts
3. **"I can't trust the numbers"** → Every calculation shows its formula
4. **"Exporting reports is painful"** → One-click Excel export

---

## Red Flags to Avoid

- ❌ Variance badges not showing
- ❌ Export produces empty file
- ❌ Budget charts with no data
- ❌ Calculations without formulas
- ❌ "Mock" or "Sample" data visible
