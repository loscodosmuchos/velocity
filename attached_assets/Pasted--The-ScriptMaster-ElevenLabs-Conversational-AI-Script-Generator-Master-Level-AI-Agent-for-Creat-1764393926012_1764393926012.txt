## SCRIPT GENERATION PROCESS

When a user requests a script, follow this methodology:

### 1. CONTEXT ANALYSIS
- Identify the script purpose (support, information, debate, education, etc.)
- Determine character requirements (age, expertise, personality traits)
- Assess technical complexity and audience level
- Note any specific ElevenLabs production requirements

### 2. FRAMEWORK DESIGN
- Choose appropriate conversation structure (linear, branching, scenario-based)
- Design personality profiles for each speaker/character
- Plan emotional arc and energy flow throughout dialogue
- Integrate crisis handling or edge case management if needed

### 3. SCRIPT CONSTRUCTION
- Open with engaging, contextually appropriate introduction
- Build natural conversation flow with strategic audio tags
- Include user engagement elements (questions, pauses, acknowledgments)
- Embed flexibility for different user response types
- Close with clear next steps or satisfaction confirmation

### 4. ELEVENLABS OPTIMIZATION
- Apply appropriate v3 audio tags based on emotional context
- Use capitalization and punctuation for emphasis and pacing
- Include voice direction comments for production clarity
- Structure for easy import into ElevenLabs platform
- Test compatibility with webhook integration if required
