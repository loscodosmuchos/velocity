## **Refine.dev: The Game-Changing Frontend Layer**

Refine.dev represents a **critical missing piece** in your VMS/ATS architecture - the **enterprise-grade React UI framework** that transforms n8n's backend automation into a production-ready, white-labeled admin platform.[^1_1][^1_2][^1_3]

### **Strategic Role in Your Stack**

**n8n (Backend Automation) + Refine.dev (Frontend Layer) = Complete VMS Platform**

Refine.dev should serve as your **primary UI/UX layer** while n8n handles workflow orchestration and business logic. This separation creates a best-in-class architecture that's both scalable and maintainable.[^1_4][^1_3]


## **ElevenLabs + MCP Immediate Implementation**

### **Step 1: MCP Server Setup (Week 1)**

**Infrastructure Requirements**[^2_8][^2_7]

```bash
# Install MCP SDK
npm install @modelcontextprotocol/sdk

# Configure OAuth 2.0 provider (Auth0 example)
- Register application in Auth0
- Obtain client_id and client_secret
- Configure callback URL: https://your-vms.com/mcp/callback
- Set scopes: read:candidates, write:placements, read:analytics
```

**MCP Server Configuration**[^2_7]

```javascript
// mcp-server.js
const { MCPServer } = require('@modelcontextprotocol/sdk');

const server = new MCPServer({
  auth: {
    type: 'oauth2',
    clientId: process.env.MCP_CLIENT_ID,
    clientSecret: process.env.MCP_CLIENT_SECRET,
    tokenEndpoint: 'https://auth.your-vms.com/oauth/token',
    scopes: ['read:candidates', 'write:placements']
  },
  resources: {
    candidates: '/api/candidates',
    jobs: '/api/jobs',
    analytics: '/api/analytics'
  },
  auditLog: true, // SOC2 compliance
  encryption: 'AES-256-GCM' // HIPAA compliance
});

server.listen(3001);
```

**n8n Workflow Logic**[^2_6]

1. Receive ElevenLabs webhook POST
2. Validate HMAC signature for security
3. Extract structured project data from `analysis.extracted_data`
4. Create record in Refine.dev backend via API
5. Trigger email notification to admin approval queue
6. If vendor project flagged, trigger contract analysis workflow
7. Store conversation transcript for audit compliance

**Stateful Conversation Setup**[^2_6]

```javascript
// Store user context between calls
app.post('/webhook/elevenlabs', async (req, res) => {
  const { data } = req.body;
  const userId = data.metadata.user_id;
  
  // Store conversation state
  await db.userStates.upsert({
    userId,
    lastConversationId: data.conversation_id,
    previousTopics: data.analysis.transcript_summary,
    projectsLogged: data.analysis.extracted_data.project_name
  });
  
  res.status(200).send('Webhook received');
});

// Next call retrieves context
async function initiateCall(userId) {
  const userState = await db.userStates.findOne({ userId });
  
  return await elevenlabs.startConversation({
    agent_id: 'velocity-project-intake',
    dynamic_variables: {
      user_name: userState.name,
      previous_projects: userState.projectsLogged.join(', '),
      previous_topics: userState.previousTopics
    }
  });
}
```


### **Step 3: Refine.dev Data Provider Connection**

**Custom Data Provider for n8n Backend**[^2_2][^2_1]

```typescript
// src/providers/n8nDataProvider.ts
import { DataProvider } from "@refinedev/core";
import axios from "axios";

export const n8nDataProvider = (apiUrl: string): DataProvider => ({
  getList: async ({ resource, pagination, filters, sorters }) => {
    const response = await axios.get(`${apiUrl}/${resource}`, {
      params: { pagination, filters, sorters }
    });
    return {
      data: response.data.items,
      total: response.data.total
    };
  },
  
  create: async ({ resource, variables }) => {
    const response = await axios.post(`${apiUrl}/${resource}`, variables);
    return { data: response.data };
  },
  
  // ... implement update, delete, getOne, etc.
});

// App.tsx
import { Refine } from "@refinedev/core";
import { n8nDataProvider } from "./providers/n8nDataProvider";

<Refine
  dataProvider={n8nDataProvider("https://n8n.your-vms.com/api")}
  resources={[
    { name: "projects", list: ProjectList, create: ProjectCreate },
    { name: "candidates", list: CandidateList },
    { name: "jobs", list: JobList }
  ]}
/>
```
